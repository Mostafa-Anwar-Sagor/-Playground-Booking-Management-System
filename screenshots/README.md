## Screenshot Guidelines

- **Format**: PNG or JPG (PNG preferred for quality)
- **Resolution**: Minimum 1280x720 pixels
- **Naming**: Use lowercase with hyphens (e.g., `user-dashboard.png`)
- **Size**: Optimize images to keep file size under 500KB each

### Windows
- Press `Win + Shift + S` for snipping tool
